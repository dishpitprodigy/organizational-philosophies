# TDA Competency Calibration

This template turns competency language into evidence. It is not a personality assessment and it is not a performance rating. It is a shared calibration tool for answering one question:

What can this person currently demonstrate, what does the next operating mode require, and what safe reps would close the gap?

## Use

1. Select the current role, target role, target scope, or next operating mode.
2. Identify the competencies that actually matter for that target.
3. Define each competency in observable behavior.
4. The employee completes a self assessment with evidence.
5. The manager completes calibration with evidence.
6. The employee and manager discuss the difference between the two reads.
7. Confirm development targets and move them into a 70-20-10 plan.

## Proficiency Stages

Use stages that describe observable work. Rename them if the organization has an existing level model, but keep the evidence discipline.

| Stage | Meaning | Evidence Pattern |
|---|---|---|
| Awareness | Understands the concept and can recognize it in context. | Can explain the idea, identify examples, and ask useful questions. |
| Guided execution | Can perform the work with support, review, or close supervision. | Completes bounded tasks when expectations, examples, and review are available. |
| Independent execution | Can perform the work reliably inside normal scope. | Delivers the work without unusual support and handles routine tradeoffs. |
| Broader coordination | Can coordinate the work across people, systems, or teams. | Anticipates dependencies, negotiates tradeoffs, and helps others execute. |
| System ownership | Can improve the operating system around the work. | Changes standards, processes, tooling, or strategy in ways that create leverage beyond direct execution. |

## Calibration Conversation

Where did the employee and manager agree?

Where did the employee rate themselves higher than the manager did?

Where did the manager rate the employee higher than the employee did?

What evidence changed either person's view?

What was based on observed behavior?

What was based on self report?

What was based on interpretation or assumption?

Which gaps matter for the current role?

Which gaps matter only for a future role?
